readme txt tokenizer

1. will tokenize a text file into its individual words and returns a list.
2. To get the list do tokenizeTxt(filename,sep) sep can be either 1 or 0. If the file is a txt of words seperated by newlines or commas then sep should be 1.
If the txt file is a natural dialogue like a page of a book then sep should be 0. 
3. The function returns a list.
